# 💻 Laptop Price Predictor (Linear Regression From Scratch) 🚀

This repository contains a comprehensive Machine Learning project focused on predicting laptop prices. Instead of using high-level libraries for modeling, the Linear Regression model was built entirely from scratch using Gradient Descent.

## 📊 Project Overview
The project explores the relationship between technical specifications (specifically RAM) and the market price of laptops. It was developed as part of my studies at the Faculty of Computers and Artificial Intelligence - Kafrelsheikh University.

## 🧠 Technical Implementation
* Mathematical Modeling: Implemented the Gradient Descent algorithm and Mean Squared Error (MSE) cost function manually.
* Exploratory Data Analysis (EDA): 
    * Scatter Plots: Visualizing the correlation between RAM and Price.
    * Pie Charts: Showing the distribution of Laptop types (Gaming, Notebook, etc.).
    * Bar Charts: Comparing average prices across different manufacturers.
* Data Preprocessing: Handled data cleaning, feature encoding, and scaling to improve model performance.

## 🛠 Tech Stack
* Language: Python 🐍
* Libraries: NumPy, Pandas, Matplotlib, Seaborn.
* Tools: VS Code, GitHub Desktop.

## 📈 Results
The model successfully predicts laptop prices with a clear regression line showing the learning progress from the initial random weights to the optimized parameters.
